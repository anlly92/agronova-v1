Terminos y Condiciones

Bienvenido al sistema integral de gestión de la finca La Elisa. Este sistema ha sido diseñado para optimizar la administración de la producción de café y cardamomo, la gestión de personal, inventarios, ventas y otros procesos administrativos. Al acceder y utilizar este sistema, usted acepta cumplir con los términos y condiciones aquí establecidos.

  • Registro y acceso
    1. Solo el Administrador Global puede registrar nuevos Administradores Secundarios.
    2. Cada administrador deberá proporcionar información veraz y actualizada al registrarse.
    3. Está prohibido compartir credenciales de acceso con terceros.
    4. Los administradores deberán mantener la confidencialidad de sus credenciales y datos.
  • Uso del Sistema
    1. El sistema debe utilizarse exclusivamente para la gestión administrativa de la finca La Elisa
    2. No está permitido ingresar información falsa, errónea o malintencionada que pueda afectar la operación del sistema.
    3. Se prohíbe el uso del sistema para actividades ilegales o no autorizadas.
  • Responsabilidades del Administrador Global
    1. Registrar y gestionar Administradores Secundarios.
    2. Asignar y administrar los permisos de acceso de cada Administrador Secundario.
    3. Velar por la correcta administración de los datos financieros, de personal y producción.
    4. Garantizar la confidencialidad y seguridad de la información del sistema.
  • Responsabilidades de los Administradores Secundarios
    1. Utilizar el sistema únicamente dentro de los permisos otorgados por el Administrador Global.
    2. Garantizar la veracidad de los datos ingresados.
    3. Reportar cualquier problema o irregularidad en el sistema al Administrador Global.
  • Seguridad y Privacidad
    1. El sistema implementa medidas de seguridad para proteger la información de accesos no autorizados.
    2. Los administradores deben utilizar contraseñas seguras y actualizarlas periódicamente.
    3. Los datos almacenados serán utilizados únicamente para los fines operativos de la finca.
